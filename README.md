 <h1 align="center">🧠 Memory App</h1>

###--

A responsive and interactive **Memory Game** built as a pure frontend project using **TypeScript** and **SCSS**.

The game is playable on both desktop and mobile devices and offers a simple, clean user experience with customizable settings and a smooth game flow.

🔗 Live Demo: https://kevin-breiter.developerakademie.net/MemoryApp/

---

## 🎮 Game Overview

The **Memory App** is a classic card matching game for two players.

### 🏁 Start Screen
When the application loads, the user lands on the start page.

- Click **Play** to begin

---

### ⚙️ Settings Screen
After clicking Play, the user can configure the game:

- 🎨 Choose a theme (game design)
- 🔵 Select starting player (Blue or Orange)
- 🃏 Choose number of cards:
  - 16 cards
  - 24 cards
  - 36 cards

After configuration, the game starts via the **Start** button.

---

### 🧩 Gameplay
- Two-player turn-based memory game
- Players flip cards and try to find matching pairs
- The game continues until all cards are revealed

---

### 🏁 Game Over Screen
- Displays the final score of both players

---

### 🏆 Result Screen
- Shows the winner or a draw
- Button to return to the start screen

---

## ✨ Features

- 📱 Fully responsive (mobile & desktop)
- 🎮 Two-player gameplay
- ⚙️ Customizable game settings
- 🃏 Dynamic card layouts (16 / 24 / 36 cards)
- 🎨 Theme selection for different game designs
- 🧭 Multi-screen navigation flow (Start → Settings → Game → Result)

---

## 🎨 Animations & UI Details

- Button hover effects
- Card rotation animation on hover
- Highlight border when matching cards are found

---

## 🛠️ Tech Stack

- **TypeScript**
- **SCSS**
- **HTML5**

> ⚠️ This is a pure frontend project. No backend or database is used.

---

## 💾 Data Storage

- No persistent storage
- No LocalStorage or database usage
- Game state resets on reload
